# Shoplists

A web application that helps you out when doing the grocery shopping. Initially this will just present you with a list of items to tick on/off, but more advanced features can be added over time. 

## Feature

### Ideas for future improvements

* Sharing lists between users
* Keeping track of what's bought
  * Attaching receipts
  * Recording detailed info per item
    * Purchase location (grocery store chain, specific store, ...)
    * Price (and price per unit)
    * When it was bought
  * Analysis over time of price, frequency, ...
* Suggestions or prediction of when an item might need to be bought again
* Comparing prices online and suggesting where to buy

## Development

This project will use Aspire for local development.
